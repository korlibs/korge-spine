# korge-spine

Spine plugin for KorGE using kproject.

## Usage

By using kproject:

### `deps.kproject.yml`

```kotlin
dependencies:
  - git::korge-spine::korlibs/korge-spine::/korge-spine::8b17996726fd373589084d8dcbcea4aa73a5c67a
```
